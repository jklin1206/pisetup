# Pi Setup

My Pi coding agent configuration, extensions, skills, and themes.

## Contents

- `AGENTS.md` — global agent instructions
- `settings.json` — Pi default settings
- `mcp.json` — MCP server config
- `extensions/` — custom extensions (Pi Workers, auto-workflow, model-router, etc.)
- `pi-workers/` — Pi Workers config + routing map
- `skills/` — custom skills (impeccable, council, wiki-librarian, obsidian-markdown)
- `themes/` — UI themes

## Usage

Clone and symlink or copy files into `~/.pi/agent/`.
